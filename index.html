Getting the Basics

After deciding I wanted to get into web development the first question on my mind was “What do I learn?” After doing some research, I ended up making my learning path based on what most entry-level web developer positions asked for, which was:

  • JavaScript
  • HTML & CSS
  • Responsive Design
  • ReactJS
  • Git
  • NodeJS

Here is how it went:

I began my Journey learning JavaScript through Udemy (paid) and W3Schools (free). If you don’t know about these, they’re great websites that allow you to learn to code by coding inside the browser. After learning JavaScript, I proceeded to learn the fundamentals of HTML & CSS and web design. After Completing JS,HTML and CSS and moved to ReactJS, I didn’t really know what exactly ReactJS was back then, but I knew everyone was talking about it, and that if I wanted to become a web developer I needed to learn it.
